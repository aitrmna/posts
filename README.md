# Shared Documents

A minimal Jekyll site for sharing documents via clean URLs.

## Usage

1. Add markdown files to `_documents/`
2. Include front matter: `title`, `description` (optional), `date` (optional)
3. Push to GitHub
4. Share the URL

## Local preview

```bash
bundle install
bundle exec jekyll serve
```

## Structure

```
_documents/     # Your documents go here
_layouts/       # Page templates
index.md        # Home page listing
_config.yml     # Site settings
```
